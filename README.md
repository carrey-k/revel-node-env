# revel-node-env
The docker image with revel and node/yarn installed based on golang docker image
# Revel Docker Images

# Main Libraries:

1. Plain: `GOLANG` and `REVEL framework`.
2. Node: `GOLANG`, `REVEL framework` and `Node`.

## Versions
|Name|Version|
|-----|------|
|GOLANG|1.14.4|
|REVEL|1.0.0|
|NodeJS|14.5.0|
